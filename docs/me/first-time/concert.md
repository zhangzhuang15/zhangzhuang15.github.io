# 第一次听演唱会 
第一次听演唱会是什么时候？很惭愧，在我27岁的时候，和老妈一起去看的。在我上学的时候，一家子只能透过电视看看演唱会的转播，如果没有转播，就看看音乐综艺，相当于听了演唱会。那时候，真的是没有那个条件，家里的收支**仅仅填饱肚子**，日子紧的时候，还要**借钱**挨过艰难的日子。

等到了我大学的时候，也勤工俭学，赚来的钱不是交学费，就是花在吃饭上边，看演出还是奢望。说实在的，见到同学去台湾或者香港做交换生，我挺羡慕的。大学暑假的时候，从朋友圈里看到同学到日本旅游，我也很羡慕。工作了两年，攒了一点钱，心里仍然**缺少安全感**，不敢乱花钱。等我工作第5个年头，我忽然有了一种想法，想要做的事情，想要的东西，不要等老了再落实，趁着年轻的时候，能办到的就给办了。恰巧那几天，我老妈随口说想看演唱会。尽管是随口说的，但我记下了。

眨眼之间，**我也不是18岁的那个高三学生了，也不是22岁刚大学毕业的学生了，我离30岁越来越近了，毫不客气地讲，我在变老，当然了，我妈也是**。说这些话是什么意思呢，就是说时间要精打细算了，***别留遗憾***。

原本我是想买张学友的北京演唱会门票。第一，演唱会在五棵松体育馆，离房山近一些，大学我也去过五棵松那边；第二，张学友不假唱，听他唱歌很划得来；第三，据说张学友会**表演大劈叉**，我觉得很有意思；第四，我妈很喜欢唱**一路上有你**这首歌。很可惜，我使用美团，头一次买票，没有什么经验，身份信息提前写好了，但是开票的时候，手速慢了，没有抢到。

后来，我听说林忆莲和张学友一样，2025年5月份也在北京开演唱会，地点是工人体育场。同样，有一些亮点吸引我。第一，**林忆莲唱功相当不错**，**不会假唱**，听她唱歌划算；第二，她唱的歌曲，我老妈也知道，**女人听女人唱歌**，更懂吧；第三，我在B站看过早些时候林忆莲的现场live视频，她高音好听，很稳，但我非常好奇，现场和视频的差距能有多大，现场版本的**高音**会不会特别**炸耳**。有了上次抢票失败的经验，我这次轻车熟路，加快手速，运气不错，抢到了两张5月24日的票。

5月24日，刚好是周六，考虑到我的**出租屋在朝阳**，家里在**房山**，我觉得周五请假回家，刚好我手头里也有一天**调休**的名额。周六晚上7点演唱会开始，我们俩下午3点出发。从工体的地铁站出来，差不多6点，到附近一个板面面馆吃点东西，我们俩就从工体北门进场了。

进场途中，并不愉快。一方面我感冒了，流鼻涕，咳嗽，嗓子哑。另一方面是工体这边太挤了。道不宽敞，人还特别多。往常说到工体，都是上学那会儿，晚上写完作业，和父母一起看北京台体育频道的国安足球直播。透过画面，感觉工体很大，非常热闹。在闸机口外边看，工体并不显得大。对了，过闸机的时候，主办方不让听众带矿泉水入场，白白浪费了我带来的两瓶水，让我着实不爽，骂了闸机口工作人员一句傻逼。

中国自打搞市场经济，都成什么鸟样了，乌烟瘴气，为了赚那点儿破钱，造成资源浪费，良好的精神风貌都是这么被破坏的。**市场经济之下，中国人的心眼儿普遍是坏的，自己人要紧防着自己人，如果你是个老实巴交的人，就一直吃亏，你吃的亏，老天爷不会给你找回来，社会不会找给你，法律不会找给你，政府不会找给你**。

以前我觉得自己就该是个善良的人，后来工作几年，去过很多地方，北京也好、天津也好、四川也罢，**你若善良，就是给坏人行方便**，这是**愚善**。所以，我变得越加谨慎，像老鼠一样，在心里反复掂量，才决定是否要行善。

话说回来，过闸机的时候，要扫描身份证，拿到纸质版的入场票。进入工体一层入口，里边就显得大了。我买的是**380元**的票，座位都在看台高层。我俩往上爬了半分钟才到地方，我妈说她累得不行。我俩买了两杯柠檬水，一杯**10块**，这10块钱花得一点儿都不值。柠檬水，就是**一杯白开水，里边放了3片儿柠檬**。

往看台走的时候，要出示一下纸质入场票。你以为这就完了？还要顺着过道儿往上爬台阶，对于我妈这个年龄段儿的人讲，台阶够抖的，我们大概是在**27层**左右。工体的座位令我非常非常失望，特别**窄**，屁股坐下去显得特别**紧**，座位之间的距离也不够，很**挤**，座位前的过道儿非常窄，只能像螃蟹一样横着挪进去。坐下之后，从上往下看，显得特别高，特别抖，特别吓人。我妈都不敢往下看，身子还要往后仰着一点，周边其他的朋友也这么觉得。我妈还特意拿手机拍了视频，**吐槽**一下——“瞅瞅这么高，真他妈吓人”。最近，北京在降温，工体里边不冷不热。

林忆莲开始演唱后，我妈的恐高感就消失了。一方面是天黑了，肉眼看不到高度落差，另一个方面是注意力全在前边的舞台上。不得不说，林忆莲的唱功比我预想得好。话筒里传出的声音，有她的**呼吸声**。高音非常稳，**非常轻松就上去了**。从大屏幕看，她唱歌的时候，脖子很放松，不像一些歌手青筋暴起。另外我比较喜欢的地方是，她唱的干净利索，不和观众互动，不说废话，一首接着一首唱歌，也不会唱着唱着就来一句“大家一起来”。开头没有什么太多老歌，我妈没听过，但我在B站听过。到了后半程，老歌一首接着一首，特别是**当爱已成往事**，**李香兰**，**至少还有你**三首歌，全场反应很激烈。我妈还跟唱了“至少还有你”。

谁知道，现在的演唱会都讲究个**callback**环节，就是歌手返场的意思。林忆莲唱完了一首歌之后，整个体育场就黑了，我以为可以离场了，然后就听到低处的人不断叫唤**林忆莲，Sandy**之类的话。有个三分钟左右吧，舞台又亮了，林忆莲再次登场。这个时候，她才和观众互动，聊几句，紧接着又唱了四首歌曲吧，有一首是**reset**，高音特别过瘾，剩下的是李宗盛时期的老歌，老歌迷很过瘾。

散场之后，老妈很开心，直呼现场版听着就是比电视上的爽。在**北门**，我们俩各自拍了照片。此时，工体外的马路已经被警察封了，我们顺着马路走到团结湖地铁口回家了。到良乡南关打了滴滴回家，到家的时候，都是**晚上12点多**了。

以前我觉得听演唱会的人矫情，不就是听歌儿吗，从电脑或者电视看视频不就完事儿了吗，干嘛花几百块钱到现场，挨挤找罪受呢？现在我必须给予改观，是的，***省钱是一种智慧，花钱也是一种智慧，有时候，钱该花就要大胆去花，不用心疼***。确实，打小儿穷怕了，特别害怕花钱，这次经历，也是给我上了一课，花钱没那么可怕，可怕的是，你没瞅准，就把钱花出去了。
